# Spotify Tracks Data Analysis using Spotify API

OVERVIEW :

-This project demonstrates end-to-end data analytics using Spotify Web API, Python, and MySQL. The system fetches real-time Spotify track metadata, processes it using Python, stores it in a relational database, and performs structured SQL analysis to uncover insights about track popularity and duration.

-The project highlights practical skills in API integration, data extraction, database design, SQL querying, and data visualization, making it suitable for data analyst and backend-oriented roles.

HIGHLIGHTS :

1. Developed a data analytics pipeline using Spotify Web API, Python, and MySQL
2. Extracted real-time Spotify track data such as track name, artist, album, popularity, and duration
3. Parsed Spotify track URLs using Regular Expressions
4. Stored processed data into a MySQL relational database
5. Performed SQL-based analysis to identify popular tracks and duration trends
6. Visualized key insights using Matplotlib
7. Enabled batch processing of multiple track URLs from a text file
8. Exported track data into CSV format for further analysis

Tech Stack :

- Python (Spotify, Pandas, Matplotllib)
- MySQL
- Spotify Web API
